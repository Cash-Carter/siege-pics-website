## Installing
1. Install [uv](https://docs.astral.sh/uv/)
2. Clone this repository
`git clone https://github.com/Cash-Carter/siege-pics-website`
3. Change to the project directory
`cd ./siege-pics-website`
4. Download the required dependencies
`uv sync`

## Adding pictures
1. Create a `siege-pics-website/site/images` directiory
2. Add photos to `images`

## Running
For running in production mode use
`fastapi.exe run`
For development mode use
`fastapi.exe dev`
